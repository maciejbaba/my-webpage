This is a sudoku solver written in JavaScript, using a backtracking algorithm to solve sudoku puzzles. The algorithm is slowed down to visualize the solving process.

Live Demo:

Key Features:

  • Backtracking Algorithm for solving Sudoku puzzles
  • Responsive design for both desktop and mobile users
  • Dark Mode enabled by default for eye comfort

Tech Stack:

HTML5, CSS3, JavaScript

How it Works:

The backtracking algorithm tries possible configurations and backtracks when a wrong configuration is found, making it effective for complex problems like Sudoku.
